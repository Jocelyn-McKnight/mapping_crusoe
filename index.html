Welcome

Welcome to the Mapping Robinson Crusoe project

This digital exhibit traces how Robinson Crusoe emerges from a dense network of earlier texts, maps, and genres. Instead of reading the novel in isolation, the site invites you to move laterally between possible sources.

Each collection gathers related materials—travel narratives, geographies, devotional works, and more. The mapping interface plots these traces onto historical geographies, highlighting how Crusoe’s imagined journeys intersect with real routes, ports, and coastlines.

Use the navigation above to learn more about the novel itself, browse the collections, and explore the interactive map that ties them together.
